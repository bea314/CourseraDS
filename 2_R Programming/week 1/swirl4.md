# 4. Vectors

Commands:
* Create a vector:  
  ```c()```
  
* Join the elements of a vector:  
  ```paste()```  
  e.g.  
  paste(my_char, collapse = " ")  
  paste("Hello", "world!", sep = " ")

* To add (or 'concatenate':  
  ```c(my_char, "word")```
